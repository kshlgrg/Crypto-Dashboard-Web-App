# 🚀 Crypto Dashboard Web App

https://kshlgrg.github.io/Crypto-Dashboard-Web-App/

## 📌 Overview

The **Crypto Dashboard Web App** is a web-based application that displays real-time cryptocurrency data using a public API. It allows users to explore different cryptocurrencies, view key market statistics, and interact with the data through search, filtering, and sorting functionalities.

This project demonstrates the use of **JavaScript, API integration, and responsive UI design**.

---

## 🎯 Objectives

* Fetch and display real-time data from a public API
* Build an interactive and user-friendly interface
* Implement search, filtering, and sorting using JavaScript
* Ensure responsiveness across devices

---

## 🔗 API Used

**CoinGecko Public API**

Endpoint:
https://api.coingecko.com/api/v3/coins/markets

### ✅ Why CoinGecko?

* Free to use (no API key required)
* Provides real-time cryptocurrency data
* Supports sorting and filtering
* Reliable and widely used

---

## ⚙️ Features

### 🔹 Core Features

* Display list of cryptocurrencies with:

  * Name
  * Symbol
  * Current Price
  * Market Capitalization
  * 24h Price Change

* 🔍 **Search**

  * Search cryptocurrencies by name or symbol

* 🎯 **Filtering**

  * Filter coins based on price change (e.g., gainers/losers)

* 📊 **Sorting**

  * Sort by price (ascending/descending)
  * Sort by market cap
  * Sort alphabetically

* 📱 **Responsive Design**

  * Works on mobile, tablet, and desktop
